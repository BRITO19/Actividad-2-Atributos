Identifica y describe las propiedades de los siguientes elementos CSS:

Tema 1-Texto

Propiedad Valor Inicial Descripción
color Depende del navegador Establece el color de letra utilizado para el texto.
font-family Depende del navegador Establece el tipo de letra utilizado para el texto.
font-size médium Establece el tamaño de letra utilizado para el texto.
font-weight normal Establece la anchura de la letra utilizada para el texto.
font-style normal Establece el estilo de la letra utilizada para el texto.
font-variant normal Establece el estilo alternativo de la letra utilizada para el texto.
font - Permite indicar de forma directa todas las propiedades de la tipografía de un texto.
text-align left Establece la alineación del contenido del elemento.
line-height normal Permite establecer la altura de línea de los elementos.
textdecoration none Establece la decoración del texto (subrayado, tachado, parpadeante, etc.).
texttransform none Transforma el texto original (lo transforma a mayúsculas, a minúsculas, etc.).
vertical-align baseline Determina la alineación vertical de los contenidos de un elemento.
text-indent 0 Tabula desde la izquierda la primera línea del texto original.
letter-spacing normal Permite establecer el espacio entre las letras que forman las palabras del texto.
word-spacing normal Permite establecer el espacio entre las palabras que forman el texto.
white-space normal Establece el tratamiento de los espacios en blanco del texto.

Característica Propiedades Descripción

font-weight

Característica Propiedades Descripción

height

Tema 4-Listas

Propiedad Valor Inicial Descripción
list-style-type disc Permite establecer el tipo de viñeta mostrada para una lista.
list-style-position outside Permite establecer la posición de la viñeta de cada elemento de una lista.
list-style-image none Permite reemplazar las viñetas automáticas por una imagen personalizada.
list-style - Propiedad que permite establecer de forma simultánea todas las opciones de una lista.
